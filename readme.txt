How to install ruby on rails setup?

--> Download rails setup from http://railsinstaller.org/en
--> Check ruby version and sqlite versions using ruby -v and sqlite3 --version on command prompt
--> run  gen install rails on cmd prompt
--> check rails --version and voila

Tutorials are in following site:
